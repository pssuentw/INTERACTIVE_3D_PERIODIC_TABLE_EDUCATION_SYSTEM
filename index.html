Help Display

Turn this help display on/off anytime. Press H or click Help.

Periodic Table Navigation

Use Table, Sphere, Helix, and Grid to change the 3D layout. Drag the table with the mouse to rotate it, use the mouse wheel to zoom, and use the side-scroll bar to move across the wider readable table.

Keyboard Controls

W move up, A move left, S move down, D move right. Use arrow keys for the 3D object lab camera. Esc closes open pages and stops voice.

Element Learning Page

Click any element box to open its detail page. It explains the element, its importance, real-life uses, discovery information, quiz question, and voice explanation.

Colour and Transparency

Use the left panel to keep original category colours, choose readable colour versions, make the table transparent, apply glass mode, or change selected/all element colours.

Game + Chatbot

Open the Game + Chatbot page for flashcard typing, element mixing, and chatbot questions. In flashcard mode, type the answer and press Enter.

3D Object Lab

Interact directly with cube, sphere, cylinder, cone, torus, plane, and pyramid. Change lighting, shadow, colour, shading, rotation, scaling, translation, speed, and camera view.
